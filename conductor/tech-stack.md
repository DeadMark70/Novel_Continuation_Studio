# Technology Stack

## Core Frameworks
- **Frontend:** React 18+
- **Meta-Framework:** Next.js (App Router)
- **Language:** TypeScript

## UI & Styling
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui (Radix UI primitives)
- **Icons:** Lucide React
- **Design Philosophy:** Noir Industrial (High contrast, sharp edges, dark mode)

## State & Data
- **Global State:** Zustand
- **Persistence:** IndexedDB with Dexie.js (for story storage and versioning)

## AI & Integration
- **LLM API:** NVIDIA NIM API
- **Capabilities:** Server-Sent Events (SSE) for real-time streaming output

## Development Tools
- **Agent:** Gemini CLI (Coding Agent)
- **Version Control:** Git
